# AGENTS.md

This repository keeps a single source of truth for AI-agent guidance at
[`CLAUDE.md`](./CLAUDE.md). Codex, Aider, Cursor, Continue, Cline, and any other
agent should read that file instead of (or in addition to) this one. The
`CLAUDE.md` filename is a historical artefact — the content is agent-neutral.

When you update one, mirror the change in the other (or update only
`CLAUDE.md` and leave this file pointing at it).
